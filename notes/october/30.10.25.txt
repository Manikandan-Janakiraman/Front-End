Destructuring:
-> Extracting values from array or objects and storing them into Variables easily
-> helps to unpack array or object data into separate variables in a clean way 

Array Destructuring:

const a = ["React", 2025,"ES6"]
const [name, year, method] = a     // Destructuring Method in array 
console.log(name);

Object Destructuring:

const obj = {name:"react", year:"2025", methods:"ES6"}
const {name, year, methods} = obj
console.log(name);


Spread Operator:
-> Used to copy or merge arrays and objects 
-> Spread Operators are unpacking the elements or unzip the elements from the arrays
and objects.

array example:
const newarray = [1,2,3]
const Spreadarray = [...newarray,2,3,4,5]
console.log(Spreadarray);


const newarray = [1,2,3]
const spretarray = [...newarray]
const [first,...datas] = newarray

object example:
const newarray = {name:"React"}
const Spreadarray = {age:"new"}

rest operator. Only you can define rest operator in function parameters.
