Media Tags & Media Queries

Media Queries:
 -> CSS Rule used to apply different styles depending on device characteristics.(Device Responsiveness)

 Screen Width/height
 Device Orientation -> Landscape, Portrait
 Resolution
 Color Schema -> dark, light screen 
 Responsive Design

Extra Small Phones -> 0 to 480px
Small, Large Phones -> 481 to 600px
Medium, tablets -> 601px to 768px
Large, laptops -> 769px to 1024px
Extra Large, desktop, tv -> 1025px to 1200px 
Ultra wide 4k display -> above 1600px 


Maximum width -> 0 to 100
Minimum width -> 100 and above 


Syntax:

@media (condition){
    selector {
        property:values;
    }
}

@media screen {

}


Example:

@media (max-width:1600px){
    body {
        background-color:#fff;

Animation: 
 ->Real frontend UI motion effects 

Transitions:
 -> Smoothly changing property value from one state to another state like hover, click, mouse drag etc...(AS)

Syntax:
 selector {
    transition: property duration timing-function delay;
 }

.btn{
    background-color:#blue;
    border:none;
    transition: all 0.5s ease;

}
.btn:hover {
    background-color:#fff;
    transform:scale(1.1)
}



Transform:
 -> Allows movement, scaling, rotation, skewing of the element.

Syntax:
selector {
    transform:function(value);
    transform:translate(x,y);
    transform:rotate(deg);
    transform:skew(x,y);
}

translate -> movement 
scale -> resize of the element 
rotate -> rotate
skew -> tilt/shear
